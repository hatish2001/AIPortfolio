====================================
CHATBOT ENVIRONMENT VARIABLES SETUP
====================================

Create a file named .env.local in the root directory with these variables:

# OpenAI API Key (Required for chatbot)
# Get your API key from: https://platform.openai.com/api-keys
OPENAI_API_KEY=sk-your-openai-api-key-here

# Pinecone API Key (Optional - for vector store)
# Get your API key from: https://app.pinecone.io/
# If not provided, the chatbot will use fallback mode with static data
PINECONE_API_KEY=your-pinecone-api-key-here

# Pinecone Index Name (Optional)
# Default: portfolio-chatbot
# When creating index in Pinecone, use:
#   - Dimensions: 1536
#   - Metric: cosine
PINECONE_INDEX_NAME=portfolio-chatbot

# Ingestion Secret (for API-based document ingestion)
# Change this to a secure random string in production
INGESTION_SECRET=change-me-in-production

====================================

QUICK START:
1. Copy the content above
2. Create .env.local file
3. Replace the placeholder values with your actual API keys
4. Restart your dev server: npm run dev
5. Test the chatbot!

See RAG_SETUP.md for detailed instructions.
